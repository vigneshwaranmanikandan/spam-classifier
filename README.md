# SMS Spam Classifier

A machine learning web app that detects spam messages with 97.5% accuracy.

## Live Demo
[Try it here](YOUR VERCEL LINK)

## Tech Stack
- Python, scikit-learn, TF-IDF, Logistic Regression
- Flask REST API
- React + Vite frontend
- Deployed on Render + Vercel

## How it works
1. Text is cleaned and tokenized
2. TF-IDF converts text to numerical vectors
3. Logistic Regression classifies as spam or ham
4. Flask API serves predictions
5. React frontend displays results
